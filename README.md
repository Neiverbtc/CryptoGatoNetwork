# CryptoGato Token - Proyecto de Contratos Inteligentes

## 🐱 Descripción

CryptoGato es un token BEP-20/ERC-20 avanzado desplegado en Binance Smart Chain (BSC) con características únicas de distribución por categorías, protección anti-whale, fees automáticos para liquidez y un sistema completo de preventa con vesting.

## 🏗️ Arquitectura del Proyecto

### Contratos Principales

1. **CryptoGato.sol** - Token principal con características avanzadas
2. **CryptoGatoPresale.sol** - Sistema de preventa con vesting
3. **CGATOLiquidityConnector.sol** - Conector multi-DEX para liquidez

### Características Principales

- ✅ Distribución controlada por categorías
- ✅ Protección anti-whale con límites configurables
- ✅ Sistema de fees automáticos para liquidez
- ✅ Preventa con múltiples fases y vesting
- ✅ Conector multi-DEX para mejor liquidez
- ✅ Funciones de seguridad y timelock
- ✅ Sistema de pausa de emergencia
